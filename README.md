# WiLL

**Voice-first input for macOS.** Speak into any app and get the right words out — properly punctuated, properly spaced, properly *yours*.

→ [will.sigworld.io](https://will.sigworld.io)

## What you get

- **Speak anywhere.** Mail, Slack, Notes, your IDE — words land clean.
- **Type smarter.** Ghost continuations as you write. One tap accepts.
- **Talk to anyone.** Live captions and on-the-fly translation across eight languages.

Built for the Mac. Made to disappear.
